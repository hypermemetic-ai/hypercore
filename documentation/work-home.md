# work-home

work-home is the home the operator's work folders — typically projects — mount into.
work-home is a child node of the root, materializing this segment; the root holds only this contract, and the home governs its own corpus.
the contract: the home mounts the operator's work folders and governs them within itself; it holds zero until the operator mounts the first.

## machine
the work-home child node is at implementation/work-home/, carrying its own documentation/ and implementation/ trees; the recursive check.sh reaches it as a node.
