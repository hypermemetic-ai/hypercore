# structure

Every governed corpus is a node.

The root corpus is a node.

A child corpus with its own `intent/` is a node.

Each node has one reserved intent tree at `intent/`.

Everything outside `intent/` is node-local material or a child node.

A child node may have no leaf material.

There is no third kind between child node and temporary effort. If temporary effort needs
a corpus boundary, it can be represented as a child node.

A node chooses its own intent segments.

The organizing document names the node's current segments and how they divide the
intent.

Parent intent reaches into a child only where a parent statement says it does.

Parent reach may be node-local, a named child, direct children, descendants, or another
named class.

An unmaterialized child slot is dormant. Its content is not invented.
